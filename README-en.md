# Awesome Game Translations [![Awesome](https://awesome.re/badge.svg)](https://awesome.re)

> A curated list of awesome community-driven video game translation, localization, and modding projects.

Community translations make games accessible across language barriers and preserve gaming heritage. This list catalogs community-driven game translation projects, organized by target language.

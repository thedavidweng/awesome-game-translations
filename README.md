## Translation Tools

General-purpose game translation tools, not tied to a single title.

- [MisakaTranslator / 御坂翻译器](https://github.com/hanmin0822/MisakaTranslator) - Real-time machine translation for visual novels and Galgames. Supports text hooking and OCR extraction, compatible with offline translation.
- [LunaTranslator](https://github.com/HIllya51/LunaTranslator) - Visual novel translator with HOOK extraction, embedded translation, emulator support (NS/PSP/PSV/PS2), OCR, LLM translation, and Japanese furigana annotation.
- [AiNiee](https://github.com/NEKOparapa/AiNiee) - One-click AI translation tool for RPGs, novels, subtitles, and documents. Deep integration with Mtool, Renpy, Translator++, ParaTranz, and more. 5800+ ★.
- [FFXIVChnTextPatch-GP](https://github.com/GpointChen/FFXIVChnTextPatch-GP) - Chinese localization tool for FINAL FANTASY XIV international client. Supports CSV-based translation and CN client asset import.

## FPS / Shooter

- [Call of Duty 4: Modern Warfare](https://github.com/thedavidweng/cod4-cn-patch) - Modern cross-platform installer for the classic 2009 Chinese localization. Pure Python, supports Windows / Linux / macOS.
- [ULTRAKILL](https://github.com/ClearwaterUK/UltrakULL) - Community translation mod supporting multiple languages. JSON-based language files for easy contribution.

## RPG

- [Deus Ex](https://github.com/fmwizard/deus-ex-i18n-toolkit) - Universal localization toolkit for Deus Ex 1 (GOTY / 1.112fm). Rewrites UE1 text packages, generates font atlases from TTF, patches DLLs for CJK word-wrap.
- [Shadowrun: Dragonfall – Director's Cut](https://github.com/fmwizard/shadowrun-dragonfall-zh) - Complete Simplified Chinese localization covering all in-game text. Steam / GOG / Epic installers.
- [Pokémon Gold / Silver](https://github.com/TomJinW/PKMN_GSCHS) - Simplified Chinese translation patch for the Korean ROM version.
- [Battle Brothers](https://github.com/shabbywu/Battle-Brothers-CN) - Chinese translation project for the tactical RPG.
- [Starsector](https://github.com/TruthOriginem/Starsector-0951-Localization) - Simplified Chinese localization for the space sim / tactical RPG.
- [Limbus Company](https://github.com/LocalizeLimbusCompany/LocalizeLimbusCompany) - Chinese localization custom language pack. 1000+ ★.

## Platforms & Communities

- [Paratranz](https://paratranz.cn) - Collaborative translation platform widely used in the Chinese game modding community. 6700+ projects.
- [Crowdin](https://crowdin.com) - Localization platform used by Minecraft, CS2, and other major games. Free for open-source projects.
- [机核 GCORES](https://www.gcores.com) - Chinese gaming culture community with frequent translation project sharing.
- [3DM 汉化](https://www.3dmgame.com) - Established Chinese game translation group.
- [游侠网](https://www.ali213.net) - One of China's largest single-player gaming communities with numerous translation teams.

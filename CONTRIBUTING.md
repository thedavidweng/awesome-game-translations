# 贡献指南

感谢你的关注！

## 添加项目

提交 Pull Request 时请包含：

1. **项目名称** — 链接到 GitHub 仓库
2. **描述** — 1–2 句中文说明
3. **语言分区** — 放在对应语言的分类下
4. **分类** — 选择合适的游戏类型分类

### 条目格式

```
- [项目名称](https://github.com/user/repo) - 简短描述。`语言`
```

### 收录标准

- 特定游戏的社区翻译补丁
- 游戏本地化相关的工具 / 框架
- 游戏文本 / 字体格式的逆向工程工具
- 社区翻译协作平台

### 不收录

- 发行商的官方本地化版本
- 无游戏相关组件的通用翻译 API
- 无可用产出的废弃项目

## 质量要求

- 项目应有清晰的 README
- 项目应可运行或有近期 Release
- 必须与电子游戏翻译 / 本地化相关

## 更新与移除

如项目已归档、废弃或链接失效，请提 Issue 或 PR 进行更新或移除。
